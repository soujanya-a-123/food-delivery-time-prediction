# 🍔 Food Delivery Time Prediction

An AI-powered web application that predicts the estimated food delivery time using Machine Learning. The application is built with **Python**, **XGBoost**, and **Streamlit**.

---

## ✨ Features

- 🚚 Predict food delivery time
- 📍 Calculate delivery distance
- 🌦 Weather & Traffic based prediction
- 🚲 Supports different vehicle types
- 📊 Delivery Status (Fast / Moderate / Slow)
- 📥 Download Prediction Report (CSV)
- 🎨 Interactive Streamlit Dashboard

---

## 🛠️ Technologies Used

- Python
- Streamlit
- XGBoost
- Pandas
- NumPy
- Scikit-learn
- Joblib

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/soujanya-a-123/food-delivery-time-prediction.git
```

### Move to Project Folder

```bash
cd food-delivery-time-prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
